if you want to create db for the project after installation
Step-by-step fix (recommended path)
1️⃣ Open your .env file

From project root:

nano .env


Verify these values:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel12_9th_httpclient
DB_USERNAME=root
DB_PASSWORD=


(Password may differ depending on your local setup)

2️⃣ Create the database (mandatory)

Login to MySQL:

mysql -u root -p


Then run:

CREATE DATABASE laravel12_9th_httpclient;
EXIT;


No tables yet — that’s fine.

3️⃣ Clear cached config (important)

Laravel aggressively caches config.

php artisan config:clear
php artisan cache:clear

4️⃣ Run migrations (creates sessions table)
php artisan migrate


You should see green success output.

5️⃣ Restart the server
php artisan serve


Open:

http://127.0.0.1:8000
