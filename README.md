- Blade Templates & Components
- Form Handling & Validation

- MySQL Database Connection
- Query Builder & Eloquent ORM
- CRUD Operations
- Sessions & Flash Messages
- File Upload
- Localization
- Pagination
- Migrations & Seeding
- Accessors & Mutators
- Eloquent Relationships
- Sending Emails
- Route Model Binding
- REST API Development
- API Authentication with Sanctum
- Laravel React Starter Kit
- Laravel 12 New Features

## Purpose
These projects were built to practice and understand Laravel concepts step-by-step through hands-on implementation.
